# Customer Analysis Project

## 📌 Project Overview

This project focuses on **analyzing customer behavior** based on multiple factors such as **age group, gender, product category, and purchasing preferences**. The goal is to extract meaningful insights that can help businesses understand their customers better and make **data-driven decisions**.

The analysis is performed using **SQL(pgAdmin4), Python, and Power BI**, making this an **Data Analytics portfolio project**.

---

## 🎯 Objectives

* Understand customer purchasing patterns.
* Analyze behavior based on age group, gender and category.
* Identify high-performing product categories.
* Visualize key insights using interactive dashboards.

---

## 🧰 Tools & Technologies Used

* **SQL (pgAdmin 4)** – Data querying, filtering, aggregations
* **Python (Jupyter Notebook)** – Data cleaning & exploratory analysis
* **Power BI** – Interactive dashboards & visualizations
* **GitHub** – Version control & project hosting

---

## 📂 Project Files

* `Customer_behavior_sql.sql`
  ➝ Contains SQL queries used for customer behavior analysis

* `Customer_behavior_code.ipynb`
  ➝ Python notebook for data exploration and preprocessing

* `customer_behavior.pbix`
  ➝ Power BI dashboard file for interesting visualization

---

## 📊 Key Insights

* Customer preferences vary significantly across **different age groups**
* Certain **product categories** contribute more to overall engagement
* Gender-based and age group -based trends highlight targeted marketing opportunities

## 🚀 How to Use This Project

1. Download or clone the repository
2. Open `.sql` file in any SQL-supported database (PostgreSQL / MySQL etc.)
3. Run the `.ipynb` file in Jupyter Notebook
4. Open `.pbix` file using **Power BI Desktop** to view dashboards
